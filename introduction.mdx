<Info>
  **Pré requisito**: Nossa API não necessita instalar nenhuma dependência. <br />
  Importante ter em mãos as credenciais e os ids que serão gerados por nossa equipe. <br />
  <p>**organizationId** - ID da organização</p>
  <p>**holderId** - ID do parceiro</p>
  <p>**planId** - ID do plano. Pode ser recuperado por API</p>
</Info>

### Acesso a nossa API

Nossa API utiliza padrão bearer token para permissão

**Etapa 1**: Autenticação:

<CodeGroup>

  ```bash cURL
    
    curl --location 'https://search.tratum.com.br/v1/login' \
    --header 'Content-Type: application/json' \
    --data-raw '{
        "email": "seu_email",
        "password": "sua_senha"
    }'

  ```
</CodeGroup>

**Etapa 1.2**: Caso seu token tenha sido expirado, você pode pegar um novo:

```bash cURL
  
    curl --location 'https://search.tratum.com.br/v1/refreshtoken' \
    --header 'Content-Type: application/json' \
    --data-raw '{
        "email": "seu_email",
        "username": "seu_username",
        "refreshToken": "seu_refresh_token"
    }'

  ```

**Etapa 2**: Consultar os planos disponíveis para sua conta:

```bash cURL
    
    curl --location 'https://search.tratum.com.br/v1/holder/{holderId}/organization/{organizationId}/balance' \
      --header 'Authorization: Bearer ************' \
      --data ''

  ```
